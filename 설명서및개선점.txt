*설명서
1. comment-login.html 을 실행시켜주시면 창이 뜹니다.
2. 로그인 전에는 좋아요및 싫어요 퍼가기 를 클릭시 반응하지않습니다.
3. 밑에 댓글 입력창을 클릭시 로그인 창이 열립니다.
4. 아이디:nam 비밀번호:1234 입력하면 다시 화면으로 넘어가며
5. 좋아요및 싫어요 퍼가기 버튼이 동작하며
6. 댓글입력란을 클릭하면 댓글 쓰기 창이 실행 댓글작성후 클릭하면
7. 도배방지로 1분간 댓글을 달수없게 설정했습니다.
8. 그리고 내가작성한 댓글은 지울수있게 설정했습니다.
9. 댓글작성시 메인창에 말풍선 갯수가 증가합니다.

*개선점 
1. 좀더 확실하게 기능을 구현하기.
2. css animation또는 color등을 추가하여 부드럽고 깔끔하게 구현하기.

*추가하고 싶은 기능 (블로그, 게시판에)
1. 동작하진 않는 사진파일이나 움직이는 이모티콘을 댓글에 작성하기 기능
2. 내용과는 상관없지만 지역 표기하기
3. 작성 시간 표기하기
4. 댓글에 음성파일 기능
5. 팔로우 또는 친구추가 기능
6. 유저에게 쪽지보내기 기능
